Wie die Flows zusammenspielen

Jede User-Aktion in einem Flow triggert eine Reaktion in einem anderen.

🔍
Seeker bucht Besichtigung
→ Lister erhält Push-Notification mit Buchungsanfrage
→ Admin sieht Aktivität im Live-Dashboard
🏠
Lister erstellt neues Listing
→ Admin-Queue erhält Review-Aufgabe
→ Nach Freigabe erscheint es auf der Seeker-Karte
🪪
Seeker schließt KYC ab
→ KI-Score ≥ 90%: Auto-Freigabe
→ Score < 90%: Admin-Review-Queue
⚠️
Schaden nach Besichtigung
→ Lister meldet Dispute
→ Admin prüft Kamera-Log & Lock-Protokoll
→ Entscheidung: Kaution einziehen oder freigeben
💶
Abschluss (Miete / Kauf)
→ Stripe zieht Provision automatisch ein
→ Lister erhält Auszahlung nach 3 Werktagen
→ Admin-Dashboard aktualisiert MRR
🗺️
Admin aktiviert neuen Markt
→ Neue Stadt erscheint auf allen Seeker-Karten
→ Lister in dieser Stadt können sofort inserieren
